What is a Linux ?


Linux is open Source unix-like and developed by community operating system it can be used for computers, servers, mainframes, mobile devices and embedded devices. It is supported on almost every computer platforms which makes it it one of the most widely supported operating systems.

In this blog, will discuss Linux and it's basic commands for a beginner to start with. Let's jump right in.



History of Linux


In 1969, Bell Lab started a project to make a common software for all the computers. The name of this software is Unix This is first operating system and It is paid.
In 1977, the Berkeley Software Distribution (BSD) was developed by the Computer Systems Research Group (CSRG), It was cheaper than UNIX.
In 1983, a programmer Richard Stallman creates the GNU project. It was an attempt at creating an UNIX type operating System but composed of entirely free Software.
In 1991, Linus Torvalds started writing its own code and developed Operating system Kernel, which he named 'Linux'.
After that Richard stallman and Linus Torvald came together & formed 'Linux' operating system. It was completely free. It is combination of Linux Kernel and GNU software. It is also called as 'GNU/Linux'.

Basic Linux Commands

Following are some basic Linux commands to get started as beginner

1) pwd [Print working directory] -

This command is used to print the present working directory.

$ pwd

2) mkdir [Make directory] -

This command is used to make an empty directory.

$ mkdir [directoryname]

3) touch -

This command is used to new create a file.

$ touch [file_name]

4) ls -

This command is used to view the contents of directory. It list all the files and folder in current directory.

$ ls

5) cd (change directory) -

This command is used to jump from one directory to another or file file to another.

$ cd [directory name] or [file name]

6) cp -

This command is used to copy files from current directory to a different directory.

$ cp [file_name] [directory_name]

7) mv -

This command is used to move files one directory to a another directory. it is also be used to rename file.

$ mv [file_name] [directory_name]

8) history -

This command will show all previous commands that have been used till now.

$ history

9) exit -

This command is used to leave the terminal.

$ exit

10) clear -

This command is used to clear the terminal.

$ clear
